# Equity Research Skills

Claude Code slash commands for fundamental equity research, powered by the
[cc-equity-research](https://github.com/prof-little-bear/cc-equity-research) toolkit
and the `drillr` MCP (SEC filings, fundamentals, earnings transcripts).

## Commands

| Command | Purpose |
|---|---|
| `/analyze TICKER` | Deep single-company research (9 lenses) |
| `/discover` | Idea generation — themes, supply chain, sector overviews |
| `/monitor TICKER` | Position tracking, thesis check, catalyst calendar |
| `/macro` | Yield curve, trade flows, labor market |

## Setup

The `drillr` MCP is configured in `.mcp.json` at the project root.
Run `/mcp` inside Claude Code to authenticate.
